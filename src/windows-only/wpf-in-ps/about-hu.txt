# Az alkamazás

> Avagy a natív PowerShell élmény.

A szoftver a következőkre nyújt megoldást:
    - Rendszer tisztítás, tisztán tartása
    - Rendszer gyorsítása vagy legalább ne lassítsuk azt.
    - A Windows a felhasználóké legyen, ne a Micro$ofté.
    - Telemetria kikapcsolás
    - Virtuális gépek optimalizálása
    - sok-sok más


Az alkalmazás felhasználói felületet nyújt a githubon megtalálható PowerShell-Lab projektemből stabil 
kiadást elérő megoldások számára.

A PowerShell-Lab jelenlegi (2023) különlegessége, hogy
100%-ban kompatibilis a windows 7-ig visszamenőleg, a gyári, beépített PowerShellel.
Továbbá konténerizálva szállít megoldást MacOS és Linux környezetben való fejlesztésre.


BACKEND:   PowerShell alapú eszközkészlet.
API:       Jelenleg JSON "adatbázist" használunk.
FRONT-END: A WPF nyújtotta előnyöket kihasználó felület.
